# Types

This directory contains the TypeScript type definitions for the project.

- `pokemon.ts`: Contains all Pokemon-related types, including:
  - `PokeDumpMon`: Raw data from PokeSnooper
  - `EnrichedPokemon`: UI-ready data with PokeAPI metadata
  - `CachedPokemon`: Cached data from PokeAPI
  - `CachedMove`: Cached move data
  - `DumpEnvelope`: The structure of the JSON dump files
